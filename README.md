# Lista 11 - IHC & UX - Aplicativo ReCiclo

Proposta de valor: Qual o principal benefício do ReCiclo para o usuário? O principal benefício é conectar pessoas que possuem objetos para doar com interessados ou entidades, facilitando o reaproveitamento de materiais e promovendo a sustentabilidade através da economia circular.

2. Descrição das TelasExplique brevemente o que acontece em cada tela desenhada. Tela 1 (Feed): O usuário visualiza os itens disponíveis em formato de lista, com foto e distância.Tela 2 (Filtros): Permite refinar a busca por categorias (móveis, roupas, etc.) e definir o raio de distância.Tela 3 (Detalhes): Exibe a descrição completa do objeto, o estado de conservação e o perfil do doador.Tela 4 (Cadastro): Formulário onde o doador insere título, descrição e anexa a foto do item.Tela 5 (Perfil/Selos): Mostra os dados do usuário e sua pontuação de gamificação ("Selos Eco") por doações feitas.


3. Jornada PrincipalDescreva o caminho do usuário desde a abertura até concluir a doação. Caminho: O usuário abre o app e acessa o Feed; ao encontrar um item de interesse, utiliza os Filtros para conferir a proximidade; clica no item para ver os Detalhes; inicia uma conversa via Chat para combinar a retirada; e, após o encontro físico, marca a doação como concluída para receber o seu Selo Eco.
